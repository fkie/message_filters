Detailed Description

template<class... Inputs>
class fkie_message_filters::TfFilter< Inputs >

Wait for TF transformations for incoming messages.

This filter is intended to be used with a Subscriber as source, and will delay incoming messages until they can be transformed to the specified TF target frames. The filter only examines the first input, even if it has multiple inputs.

namespace mf = fkie_message_filters;
mf::Subscriber<M> sub;
mf::TfFilter<mf::Subscriber<M>::Output> flt;
ros::NodeHandle nh;
sub.subscribe(nh, "topic", 10);
flt.init(tf2_buffer, "target_frame", 10, nh);
mf::chain(sub, flt);
